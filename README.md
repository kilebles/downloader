# Автоматизация televzr

Скрипт для автоматической загрузки видео из списка в приложение televzr.

## Установка

1. Установите Python 3.7 или новее

2. Установите зависимости:
```bash
pip install -r requirements.txt
```

## Использование

### Базовое использование

1. Запустите приложение **televzr**
2. Установите курсор в поле для ввода ссылки
3. Запустите скрипт:
```bash
python main.py
```
4. У вас будет 5 секунд, чтобы переключиться на окно televzr
5. Скрипт автоматически обработает все ссылки из `videos.txt`

### Настройка

В начале файла `main.py` есть настройки, которые можно изменить:

```python
DELAY_AFTER_PASTE = 1           # Задержка после вставки ссылки (сек)
DELAY_AFTER_SEARCH = 5          # Задержка после поиска (сек)
WAIT_FOR_DOWNLOAD_BUTTON = 10   # Время ожидания кнопки "Скачать" (сек)
DELAY_AFTER_DOWNLOAD = 2        # Задержка после нажатия "Скачать" (сек)
DELAY_BETWEEN_VIDEOS = 1        # Задержка между видео (сек)
INITIAL_DELAY = 10              # Время на переключение окна (сек)

# Метод работы с кнопкой "Скачать"
USE_TAB_FOR_DOWNLOAD = True     # Использовать Tab для перехода к кнопке
TAB_COUNT = 1                   # Количество нажатий Tab
```

**Важные параметры:**
- `WAIT_FOR_DOWNLOAD_BUTTON` - время, которое скрипт ждет появления кнопки "Скачать" после поиска
- `TAB_COUNT` - подберите это значение экспериментально (обычно 1-3)

### Настройка координат кнопок (опционально)

Если автоматическое нажатие Enter не работает, можно настроить координаты кнопок:

1. Запустите этот код, чтобы узнать координаты мыши:
```python
import pyautogui
import time

print("Наведите курсор на кнопку через 3 секунды...")
time.sleep(3)
print(f"Координаты: {pyautogui.position()}")
```

2. Запишите координаты и укажите их в `main.py`:
```python
SEARCH_BUTTON_POS = (800, 400)    # Координаты кнопки "Поиск"
DOWNLOAD_BUTTON_POS = (900, 500)  # Координаты кнопки "Скачать"
```

### Подбор количества Tab

Чтобы определить правильное значение `TAB_COUNT`:

1. Вставьте тестовую ссылку в televzr вручную
2. Нажмите Enter для поиска
3. Подождите появления кнопки "Скачать"
4. Нажимайте Tab и считайте, сколько раз нужно нажать, чтобы попасть на кнопку "Скачать"
5. Установите это значение в параметр `TAB_COUNT`

Если Tab не работает, установите `USE_TAB_FOR_DOWNLOAD = False` и настройте координаты кнопки.

## Остановка скрипта

Нажмите `Ctrl+C` в консоли для остановки скрипта.

## Важно

- Не двигайте мышь и не переключайте окна во время работы скрипта
- Убедитесь, что окно televzr активно перед началом работы
- Скрипт работает в том разрешении экрана, в котором он был настроен
